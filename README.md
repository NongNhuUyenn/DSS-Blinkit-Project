\# DSS Blinkit Project



This repository contains the implementation for Member 3's part of the DSS Blinkit project.



\## Main Objectives



This part focuses on two decision-support tasks:



1\. Forecasting daily revenue and daily order volume.

2\. Predicting delivery delay risk for each order.



\## Input Data



The project uses the following datasets:



\- `blinkit\_orders\_clean.csv`

\- `blinkit\_marketing\_performance\_clean.csv`

\- `blinkit\_data2/blinkit\_order\_items.csv`

\- `blinkit\_data2/blinkit\_products.csv`

\- `blinkit\_data2/blinkit\_customers.csv`

\- `blinkit\_data2/blinkit\_delivery\_performance.csv`
